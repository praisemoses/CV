# Personal CV Website

A responsive, accessible personal CV website built with semantic HTML and CSS.
Designed with a retro, pixel-inspired aesthetic while maintaining professional
readability and accessibility standards inspired by https://roadmap.sh/projects/single-page-cv. 

## Overview

This project serves as my online curriculum vitae, showcasing my education,
skills, and experience in a clean, minimal, and mobile-friendly format.
The site emphasizes semantic HTML, accessibility best practices, and modern CSS
techniques.

## Features

- Semantic HTML5 structure
- Responsive layout for desktop and mobile
- Accessible typography and color contrast
- Reduced-motion friendly animations
- Retro pixel-style design
- SEO and Open Graph meta tags
- Custom favicon

## Built With

- HTML5
- CSS3
- Google Fonts (Pixelify Sans)
